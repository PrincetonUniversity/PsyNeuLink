Primer
======

PsyNeuLink's Primer provides practical tutorials focused on building, configuring, and running models using PsyNeuLink. These tutorials emphasize the core technical capabilities of PsyNeuLink, guiding users step-by-step through its features.

For tutorials that place PsyNeuLink within a broader theoretical context, see the :doc:`Computational Neuroscience Course </tutorials/computational_neuroscience_course/index>`. That course explores computational concepts and methods central to modern cognitive neuroscience, providing practical applications using PsyNeuLink.
